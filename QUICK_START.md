## ğŸ“… Managing Bookings

### View Bookings
1. Go to Admin Dashboard
2. Click "Tempahan" (Bookings) tab
3. Filter by date or status
4. Click booking to see details

### Block Dates
1. Go to Admin Dashboard
2. Click "Kalendar" (Calendar) tab
3. Click date to block/unblock
4. Red dates = blocked/booked

### Send WhatsApp Messages
1. Click booking in Tempahan tab
2. Select WhatsApp template
3. Click "Hantar" (Send)
4. Message opens in WhatsApp

## ğŸš¨ Troubleshooting

**Booking not appearing?**

- Verify booking was submitted
- Refresh page

**Admin login not working?**
- Clear browser cache
- Check caps lock
- Use password reset if forgotten

- Check internet connection
- Wait 30 seconds for sync

**WhatsApp not opening?**
- Ensure WhatsApp is installed
- Check phone number format
- Try again in 5 seconds
